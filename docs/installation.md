# Installation

Install Jest using your favorite package manager:

<codeblock storageKey="package-manager">

```bash title="npm"
npm install -g @wocker/cli
```

```bash title="yarn"
yarn global add @wocker/cli
```

```bash title="pnpm"
pnpm add -g @wocker/cli
```

</codeblock>

## Project initialisation

```shell
cd /project-dir
```

```shell
ws init
```
