# Installation

Install Wocker using your favorite package manager:

<codeblock storageKey="package-manager">

```bash title="npm"
npm install -g @wocker/ws
```

```bash title="yarn"
yarn global add @wocker/ws
```

```bash title="pnpm"
pnpm add -g @wocker/ws
```

</codeblock>
