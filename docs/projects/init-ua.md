# Ініціалізація проєкту

## Створення проєкту

Спочатку, потрібно перейти у відповідну директорію проєкту:

```shell
$ cd ~/project-dir
```

У відповідній директорії `ws init`

```shell
$ ws init
? Project name: (example)
```

```shell
? Project type:  (Use arrow keys)
❯ Image
  Dockerfile
  Preset
```
